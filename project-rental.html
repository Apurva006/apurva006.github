SQL — Core Queries Written
Written from scratch · PostgreSQL-compatible
-- Query 1: County-level rent inflation vs 2019 baseline
WITH base_2019 AS (
  SELECT county, AVG(standardised_rent) AS avg_rent_2019
  FROM  rtb_rents
  WHERE year = 2019
  GROUP BY county
),
current_yr AS (
  SELECT county, AVG(standardised_rent) AS avg_rent_2024
  FROM  rtb_rents
  WHERE year = 2024
  GROUP BY county
)
SELECT
  c.county,
  ROUND(b.avg_rent_2019, 0)                           AS rent_2019,
  ROUND(c.avg_rent_2024, 0)                           AS rent_2024,
  ROUND((c.avg_rent_2024 - b.avg_rent_2019) / b.avg_rent_2019 * 100, 1) AS pct_change
FROM current_yr c
JOIN base_2019   b ON c.county = b.county
ORDER BY pct_change DESC;

-- Query 2: Affordability flag — counties above national median
SELECT
  county,
  ROUND(AVG(standardised_rent), 0) AS avg_rent,
  CASE
    WHEN AVG(standardised_rent) > (SELECT PERCENTILE_CONT(0.5)
         WITHIN GROUP (ORDER BY standardised_rent) FROM rtb_rents WHERE year = 2024)
    THEN 'Above Median'
    ELSE 'Below Median'
  END AS affordability_flag
FROM rtb_rents
WHERE year = 2024
GROUP BY county
ORDER BY avg_rent DESC;

📈
Commuter belt rising faster than Dublin itself
Kildare (+52%) and Wicklow (+53%) have outpaced Dublin (+47%) in 5-year rent growth. Remote work has pushed demand outward — supply hasn't followed.
🏠
3-bed family homes most supply-constrained
SQL analysis shows 3-bed properties have the widest gap between demand (time on market) and supply (new listings). Average 3-bed Dublin rent hit €2,680 in 2024.
💡
Connacht counties offer 60%+ cost saving
Leitrim and Roscommon average under €850/month — 60% below Dublin. Full remote workers represent a growing segment relocating to these counties.
📊
Q1 consistently shows highest annual increases
RTB data shows Q1 is when the largest annual rent increases are registered across all counties — landlords time lease renewals with January demand peaks.
