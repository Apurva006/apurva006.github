Excel & Tableau Methodology Tools Applied
Excel — Data Cleaning & Analysis
  • Power Query used to consolidate 12 monthly supplier delivery files into one structured table
  • VLOOKUP to cross-reference supplier IDs with product category master data
  • Pivot tables to aggregate waste cost by store, category, week and supplier
  • Conditional formatting to flag stores with waste > 5% of revenue (red threshold)
  • SUMIF and COUNTIF formulas to calculate spoilage rates per delivery delay band
Tableau — Dashboard Delivery
  • Connected Tableau directly to the cleaned Excel model as live data source
  • Built 4 interactive dashboard views: by category, by store, by supplier, by month
  • Calculated fields for waste % of revenue and YoY change per category
  • Colour-coded threshold indicators — green/amber/red — for category manager use
  • Dashboard filter actions allowing drill-down from national to individual store level

🥦
Fresh Produce + Bakery = 34% of all waste cost
Excel pivot analysis shows these two categories account for €480K of the €1.24M total. Both are driven by over-ordering — the ordering model needs recalibrating against actual sell-through rates.
🚚
Late deliveries cause 42% of all Dairy spoilage
VLOOKUP cross-referencing supplier delivery times with spoilage records revealed a clear pattern: when deliveries arrive more than 4 hours late, spoilage in Dairy rises by 42%. 3 suppliers account for 78% of late deliveries.
🏪
3 stores generate 51% of total shrinkage
Store-level Tableau analysis identified 3 outlets with shrinkage rates more than double the chain average. Common factor: all 3 have the oldest CCTV systems and highest footfall. Security investment ROI is strong.
🕐
Bakery markdown timing recovers €82K
Simulating a 3pm markdown on same-day bakery items (vs current 5pm) in the Excel model showed a projected sell-through increase from 62% to 88% — recovering an estimated €82K annually in end-of-day waste.
Business Recommendations
Delivered as a Tableau dashboard + Excel cost model for operations management
01
Dynamic Ordering Model — Produce
Replace fixed weekly order volumes with sell-through-based dynamic forecasting. Projected waste reduction: 35% in Fresh Produce. Recovers €98K.
02
3pm Bakery Markdown Rule
Implement same-day markdown at 3pm. Excel model shows sell-through improves from 62% → 88%. Recovers €82K annually in end-of-day waste.
03
Supplier SLA Enforcement
3 suppliers account for 78% of late deliveries. Introduce delivery SLA with penalty clauses. Projected Dairy spoilage reduction: 42%. Recovers €64K.
04
Security Upgrade — 3 Stores
CCTV and staffing investment in 3 high-shrinkage stores. Estimated cost €48K. Projected shrinkage reduction: 60%. Payback period: 8 months.
